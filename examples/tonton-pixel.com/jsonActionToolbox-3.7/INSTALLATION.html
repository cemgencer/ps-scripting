Installation of JSON Action Toolbox

“JSON Action Toolbox” is available in two formats:

  • A Classic/Carbon plug-in file originally written with MPW (Macintosh Programmer’s Workshop) and the Photoshop 6.0 SDK. It has been successfully tested in Photoshop 6.0.1 and Photoshop 7.0 in Classic environment, and in Photoshop 7.0 and Photoshop CS on Mac OS X. It should also work in Photoshop CS2.
  • A Universal Binary plug-in folder made of individual modules built with the Xcode 3 IDE and the Photoshop CS4 SDK. It has been successfully tested in Photoshop CS4 on Mac OS X, and should work in CS3 and CS5 (but only in 32-bit mode) too.

The archive is made of two main folders, one for each format of the plug-in:

  • Classic-Carbon

    • jsonActionToolbox

  • Universal Binary

    • jsonActionToolbox folder
      • jsonActionsFileParser.plugin
      • jsonEavesdropper.plugin
      • jsonEventListener.plugin
      • jsonIDLister.plugin
      • jsonPropertyGetter.plugin

The adequate plug-in format should be placed in an appropriate location somewhere inside the "Plug-Ins" folder, for instance in the "Automate" folder of the "Adobe Photoshop Only" folder. After (re)starting Photoshop, the respective dialog windows of the different modules can be accessed the usual way from the "File>Automate" sub-menu. On request, the respective log files will be automatically created in a folder named "JSON Action Toolbox Logs Folder" located on the Desktop.

The JSON Action Toolbox software is distributed as Freeware.
